# @keepass-web/kdbx

KDBX 3.1 and 4.x parser and serializer for the [keepass-web](https://github.com/keepass-web) project.

Handles KDBX files reporting 0x67 format.

## Specification

See [SPEC.md](./SPEC.md).

## Usage

Published to npm under the `@keepass-web` scope with provenance signing.

## Development

```sh
npm ci
npm run typecheck
npm run lint
npm test
```
